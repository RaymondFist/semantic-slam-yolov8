# Semantic-Aware Stereo Visual-Inertial SLAM with YOLOv8

This repository contains the complete source code and implementation details for the **semantic-aware stereo visual-inertial SLAM system** proposed in our paper. The system integrates YOLOv8 for real-time object detection and semantic feature extraction, significantly improving pose estimation and map consistency in dynamic environments.

**Key Features:**
- Full stereo + IMU fusion with semantic gating
- Pre-trained YOLOv8 weights (COCO / custom fine-tuned)
- Configuration files for KITTI and EuRoC datasets
- Evaluation scripts to reproduce all figures and tables from the paper
- MIT license – free for VR/AR research and development

**Reproducibility:** All experimental results presented in the paper can be regenerated using the provided code and datasets.
